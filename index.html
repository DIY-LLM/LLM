--- README.md --- # DIY LLM Toolkit (GitHub Pages ML Offload) This repository hosts a complete, client-heavy, **static web application** designed for **GitHub Pages** deployment. It implements a hybrid machine learning training architecture: 1. **Local Mode (`config.js: USE_CLOUD_TRAINING: false`):** Trains a small **LSTM model using TensorFlow.js** entirely in the browser. 2. **Cloud Offload Mode (`config.js: USE_CLOUD_TRAINING: true`):** Connects securely to **AWS SageMaker** via the browser-compatible AWS SDK to orchestrate and monitor heavy **Transformer Encoder** training jobs. All AWS interactions are client-initiated; **no backend services are hosted in this repo.** ## üöÄ Quick Start 1. **Local Run:** Serve the repository root using any static web server (e.g., `npx http-server .` or Python's `http.server`). 2. **GitHub Pages:** Commit and push to GitHub. Ensure the Pages source is set to the repository root. 3. **Run Tests:** Open `test/tests.html` in your browser to verify module integrity. 4. **Security Check:** Run `./scripts/check_no_aws_keys.sh` to confirm no hardcoded credentials are present. ## ‚öôÔ∏è Configuration & AWS Setup To enable cloud offloading, edit the **`config.js`** file with your AWS details (using **ALL CAPS** placeholders): ### 1.
